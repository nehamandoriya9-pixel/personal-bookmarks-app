# Personal Bookmarks App

A full-stack bookmark management application built with Next.js, Supabase, and Resend.

## Features

- User authentication
- Bookmark CRUD
- Public and private bookmarks
- Public profiles

## Tech Stack

- Next.js
- Supabase
- Resend
- Tailwind CSS
- Vercel

## Local Development

Install dependencies:

```bash
npm install

```

Start the development server:

```bash
npm run dev

```

Open:

```text
http://localhost:3000

```
